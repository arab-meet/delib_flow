.. _navigation:

Navigation
###########

This is a navigation tutorial, you can get started with

1. Run simulation with

  .. code-block:: bash

    ros2 launch tiago_sim tiago_sim.launch.py

2. Create a map

  .. code-block:: bash

    ros2 launch tiago_slam tiago_slam_toolbox.launch.py

3. Start navigation

  .. code-block:: bash

    ros2 launch tiago_nav tiago_nav2.launch.py
